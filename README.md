# Pendant!

This is my submission to Hack Club Pendant YSWS!

I designed a small board with a rotary encoder, 2 push-buttons, an OLED screen and a XIAO ESP32! It can be used for games, internet tasks with web APIs, Home Assistant control and more!

For part details, see [the BOM](https://github.com/duckida/pendant/blob/main/bom.md). To learn about how I designed this, check out the [journal](https://github.com/duckida/pendant/blob/main/journal.md)!
